Work

Pley is a single-page web app inspired by Yelp that allows users to search for and review businesses in their area. Pley was built with Rails on the backend with a React/Redux front end. With a dynamic search, Pley will search for businesses with "like" names or tags based on the inputted query. Each business has its own show page that allow the user to write a review, update, and delete a review for each business.

Super Jeff is a game similar to Mario built using JavaScript and Easel. Super Jeff has built in physics via seamless collision-detection that give the user the real feel for Mario. Jeff utilizes the native DOM API to handle events happening on the window that works seamlessly with a vanilla-JS state machine that controls which way Jeff is facing, walking, or jumping.
